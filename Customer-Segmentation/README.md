# Customer Segmentation Project

## Project Overview
Customer segmentation is the process of dividing customers into groups based on similar characteristics or purchasing behavior.

This project analyzes customer data to identify different customer groups.

## Objectives
- Understand customer purchasing patterns
- Group customers with similar behavior
- Identify potential target customers for marketing

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
