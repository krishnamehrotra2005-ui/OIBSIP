# Exploratory Data Analysis on Retail Sales

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to discover patterns, trends, and customer behavior.

## Dataset Features
- Transaction ID
- Date
- Customer ID
- Gender
- Age
- Product Category
- Quantity
- Price per Unit
- Total Amount

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Files
- EDA_retail.ipynb
- EDA_on_retail.pdf
- retail_sales_dataset.csv
